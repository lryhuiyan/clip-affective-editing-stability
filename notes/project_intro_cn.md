# 基于 CLIP 的反事实图像情绪编辑稳定性分析

## 一、项目背景

图像情绪理解是计算机视觉中一个比较有意思的方向。相比于普通的图像分类，图像情绪并不只取决于画面中有什么物体，也会受到亮度、色彩、对比度、饱和度、色温等视觉因素影响。

例如，一张偏暗、低饱和、冷色调的图片可能更容易让人联想到孤独或压抑；而一张高亮度、高饱和、暖色调的图片可能更容易让人联想到积极或浪漫的氛围。

但是，不同情绪是否都能通过固定的视觉编辑方式稳定增强？比如，是否只要提高亮度和饱和度，就一定能让图片更“happy”？是否降低亮度和饱和度，就一定能让图片更“lonely”？这个问题并不确定。

因此，本项目尝试使用 CLIP 模型作为图文相似度评价工具，对不同情绪目标下的图像编辑效果进行实验分析。

## 二、研究问题

本项目关注的问题是：

> 固定的低层视觉属性编辑方式，是否能在不同图片和不同情绪目标下稳定提升图像的情绪匹配程度？

更具体地说，本项目希望分析：

1. 哪些情绪更容易通过亮度、对比度、饱和度、色温等视觉属性增强？
2. 哪些情绪的最佳编辑方式更加稳定？
3. 是否存在某些情绪或图片不适合使用固定编辑预设？
4. 图像情绪编辑的效果是否同时依赖于目标情绪和图片本身？

## 三、方法设计

本项目没有直接使用生成式模型重新生成图片，而是采用更轻量、可解释的反事实编辑方法。

对于每一张输入图片和每一种目标情绪，程序会生成以下几个版本：

1. 原图 original
2. 仅调整亮度 brightness_only
3. 仅调整对比度 contrast_only
4. 仅调整饱和度 saturation_only
5. 仅调整色温 temperature_only
6. 同时调整所有属性 all_attributes

每一种情绪对应一组人工设计的视觉编辑参数。例如，lonely 更倾向于降低亮度、降低饱和度和偏冷色调；scary 更倾向于降低亮度、提高对比度并偏冷色调；romantic 则更强调暖色调。

随后，项目使用 CLIP 计算每个图片版本和目标文本之间的相似度。目标文本包括：

```text
a happy image
a lonely image
a scary image
a peaceful image
a romantic image
a hopeful image
```

对于每一个“图片-情绪”组合，CLIP 分数最高的版本被认为是该组合下的最佳编辑结果。

评价指标为：

```text
best_improvement = best_score - original_score
```

也就是最佳编辑版本相比原图的 CLIP 相似度提升。

## 四、实验设置

实验共使用：

* 100 张输入图片
* 6 种目标情绪：happy、lonely、scary、peaceful、romantic、hopeful
* 每张图片对每种情绪生成 6 个版本
* 最终得到 600 条图像-情绪实验结果

实验输出包括：

1. 每个情绪的平均分数提升
2. 每个情绪下不同编辑方式成为最佳的次数
3. 每个情绪的稳定性分析
4. 若干典型案例图像和对应分数

## 五、实验结果

从平均提升来看，lonely 和 scary 的提升最明显。

实验结果如下：

| 情绪       |  平均最佳提升 |
| -------- | ------: |
| lonely   | 0.01185 |
| scary    | 0.00916 |
| romantic | 0.00420 |
| peaceful | 0.00345 |
| happy    | 0.00312 |
| hopeful  | 0.00224 |

其中，lonely 的平均提升最高，说明当前这套低层视觉属性编辑方式最容易增强 CLIP 对“孤独感”的匹配判断。scary 的平均提升排名第二，也说明恐怖感与低亮度、高对比度、冷色调等视觉属性有较强关系。

相比之下，happy 和 hopeful 的平均提升较低，说明正向情绪不一定能够通过简单的亮度、饱和度、色温编辑稳定增强。它们可能更依赖图片中的语义信息，例如人物表情、场景内容、动作和物体关系等。

## 六、稳定性分析

为了分析不同情绪下编辑方式是否稳定，本项目统计了每种情绪中出现次数最多的最佳编辑方式。

| 情绪       | 最常见最佳编辑方式        |       次数 |   比例 |
| -------- | ---------------- | -------: | ---: |
| lonely   | all_attributes   | 79 / 100 | 0.79 |
| scary    | all_attributes   | 64 / 100 | 0.64 |
| peaceful | all_attributes   | 43 / 100 | 0.43 |
| romantic | temperature_only | 37 / 100 | 0.37 |
| hopeful  | all_attributes   | 29 / 100 | 0.29 |
| happy    | all_attributes   | 25 / 100 | 0.25 |

结果显示，lonely 的稳定性最强。在 100 张图片中，有 79 张图片在 lonely 目标下的最佳编辑方式都是 all_attributes。这说明对于 lonely 情绪，亮度、饱和度、色温等属性的组合编辑具有较高一致性。

scary 也表现出较强稳定性，all_attributes 在 64 张图片中成为最佳编辑方式。

但是 happy 和 hopeful 的最佳编辑方式分布比较分散，没有一种编辑方式能够稳定占优。这说明正向情绪的编辑效果更加依赖图片本身，固定预设不一定有效。

## 七、典型案例分析

为了进一步理解实验结果，本项目选取了若干典型案例。

在 test_027 的 lonely 目标下，all_attributes 编辑将 CLIP 相似度从 0.2075 提升到 0.2382，提升幅度为 0.0307。这是所有 lonely 样本中提升最明显的案例之一，说明组合属性编辑可以明显增强孤独感。

在 test_054 的 scary 目标下，all_attributes 编辑将分数从 0.2130 提升到 0.2365，提升幅度为 0.0235。这说明恐怖感也可以通过多种低层视觉属性的组合调整得到增强。

在 test_085 的 romantic 目标下，最佳编辑方式是 temperature_only，分数从 0.2225 提升到 0.2352，提升幅度为 0.0127。这说明 romantic 情绪可能对暖色调更加敏感。

同时，也存在一些反例。例如，在 test_001 的 happy 目标和 test_029 的 hopeful 目标下，original 原图本身就是最佳版本，编辑后并没有进一步提升 CLIP 分数。这说明固定视觉编辑并不总是有效，有些图片本身已经更符合目标情绪。

## 八、主要发现

本项目得到的主要结论包括：

1. 固定情绪编辑预设并不是对所有情绪都稳定有效。
2. lonely 和 scary 更容易通过低层视觉属性组合编辑增强。
3. romantic 对色温变化比较敏感。
4. happy 和 hopeful 的最佳编辑方式更分散，平均提升也更低。
5. 部分图片原图已经是最优版本，继续编辑反而无法提升情绪匹配。
6. 图像情绪编辑效果同时具有情绪依赖性和图像依赖性。

因此，本项目更适合被理解为一个“情绪编辑稳定性分析实验”，而不是一个简单的情绪滤镜工具。

## 九、项目局限性

本项目也存在一些明显局限。

首先，实验使用 CLIP 相似度作为评价指标，因此结果反映的是 CLIP 的图文匹配判断，并不完全等同于人类真实的情绪感知。

其次，当前编辑方式只包含亮度、对比度、饱和度和色温等低层视觉属性，没有涉及人物表情、物体内容、场景结构等高级语义变化。

第三，情绪编辑参数目前是人工设定的，而不是通过优化算法自动学习得到的，因此还不具备真正的自适应能力。

最后，实验规模为 100 张图片，适合作为小规模探索实验，但如果要得到更强的结论，还需要更大规模、更有控制的数据集以及人工评价实验。

## 十、未来改进方向

后续可以从以下几个方面继续改进：

1. 增加更多目标情绪，例如 sadness、anger、calm、warm 等。
2. 使用不同 CLIP 模型进行对比，观察结果是否一致。
3. 引入人工评价，比较 CLIP 分数和人类情绪感知是否一致。
4. 将固定参数编辑改为图像自适应编辑。
5. 使用优化算法自动寻找每张图像的最佳编辑参数。
6. 加入语义编辑方法，例如改变表情、天气、场景元素等。
7. 做成一个可交互 Demo，用户上传图片并选择目标情绪，系统输出不同反事实版本和 CLIP 分数分析。

## 十一、项目总结

本项目实现了一个基于 CLIP 的反事实图像情绪编辑稳定性分析框架。通过对 100 张图片和 6 类情绪目标进行批量实验，项目发现不同情绪对低层视觉编辑的响应差异明显。

其中，lonely 和 scary 更容易通过组合视觉属性编辑获得稳定提升，而 happy 和 hopeful 的编辑效果较弱且不稳定。这说明图像情绪编辑并不能简单依赖固定滤镜，而需要考虑目标情绪和图像内容之间的关系。

整体来看，本项目的价值不在于声称解决了图像情绪编辑问题，而在于提供了一个可解释、可复现的小型实验框架，用来分析不同情绪下低层视觉属性编辑的有效性和稳定性。
